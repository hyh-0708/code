In this experiment, we used an Infrared (IR) Receiver module in combination with a buzzer, aiming to build a basic obstacle detection and response system using the Arduino Uno.
Arduino Uno (Elegoo Edition) Serves as the microcontroller, responsible for receiving and processing the signal data from the IR receiver. Infrared Receiver The small black three-legged component placed on the breadboard. It is used to receive infrared signals from a remote control. When a button on the remote is pressed, the IR receiver captures the signal and sends data to the Arduino for interpretation. Resistor Likely used to pull down or pull up one of the IR receiver pins (often the output signal pin) to stabilize the signal or limit current as needed. Breadboard and Jumper Wires Used for non-permanent circuit construction and easy wiring between components. USB Cable and Computer Used to upload Arduino code and provide power to the board.

This isÂ  actual setup of a simple distance-based alarm system built with a microcontroller, a distance sensor, and a buzzer on a breadboard.
Microcontroller Board Connected to a laptop via USB to receive power and upload code. Breadboard Used to arrange and connect the components without soldering. Distance Sensor Positioned on the right side of the breadboard, it detects how far an object is from the sensor using sound waves. Buzzer A black circular component that emits sound when triggered, used as an alert signal. Resistor Placed in the circuit to control current or stabilize signal input/output. Jumper Wires Various colored wires that connect the components to the microcontroller and complete the circuit.
The sensor continuously measures the distance between itself and any nearby object. When an object comes too close, the buzzer is triggered and produces a sound. Once the object moves away and the distance increases, the buzzer stops. This hands-on experiment demonstrates how input data from a sensor can be used to control an output device, helping us understand basic interactive system design.
In a later experiment, we explored a more creative and interactive way to control input by using natural conductive materials. We used a potato, a lemon, and aluminum foil as electrodes to detect bioelectric signals from the human body. Materials Used:
Potato and Lemon: These fruits and vegetables contain moisture and electrolytes, making them weak conductors of electricity. Aluminum Foil: Used as a low-cost electrode or touch surface to enhance conductivity and allow easy contact with the human skin. Arduino Uno: Acts as the microcontroller to measure the signal and respond accordingly. Buzzer: Used as the output device, triggered when the signal is detected.
